# 各ウィンドウ用プロンプト（コピペ用）

4つの別ウィンドウでClaude Codeを開き、それぞれ以下をコピペして実行してください。

---

## 🟡 ウィンドウ1: ココナラ調査

```
ココナラで「マッチングアプリ用プロフィール写真撮影サービス」の市場調査をして。

【調査してほしいこと】
1. 競合サービス一覧（15-20件）
   - サービス名、価格、販売実績数、評価

2. 価格帯分析
   - 最安値〜最高値
   - 売れ筋価格帯

3. サービス内容の比較
   - 撮影枚数、レタッチ有無
   - スタジオ/出張/データのみ
   - オプション内容

4. レビュー分析
   - 高評価の共通点
   - 低評価・クレームの傾向
   - ユーザーが本当に求めているもの

5. 売れてる出品者の特徴
   - プロフィールの書き方
   - サムネイル画像の傾向

【検索キーワード】
- マッチングアプリ 写真
- プロフィール写真 撮影
- マッチングアプリ 写真 男性

【出力】
c:\re\カメラサービスの比較\01_coconala.md に保存して。
最後に「ビジネスへの示唆」を3-5点まとめて。
```

## 🟢 ウィンドウ2: タイムチケット調査

## 🔵 ウィンドウ3: Twitter/X調査

```
Twitter/Xで「マッチングアプリの写真」に関するユーザーの生の声を調査して。

【調査してほしいこと】
1. ユーザーの悩み・不満
   - 「マッチングアプリ 写真 撮れない」
   - 「マッチングアプリ 写真 難しい」
   - 「マッチングアプリ いいね こない」
   で検索して傾向を把握

2. 撮影サービス利用者の声
   - 「プロフィール写真 撮ってもらった」
   - 満足/不満の声

3. 潜在ニーズの発掘
   - 「こんなサービスあったら」
   - 「写真さえ良ければ」
   - 「誰か撮ってほしい」

4. ターゲット像の把握
   - どんな人が困っているか
   - 男性/女性の悩みの違い

5. 心理的障壁
   - なぜサービスを使わないのか
   - 恥ずかしさ、価格、その他

【出力】
c:\re\カメラサービスの比較\03_twitter.md に保存して。
（個人情報は含めず、傾向の要約で）
最後に「ビジネスへの示唆」を3-5点まとめて。
```

## 🟣 ウィンドウ4: YouTube調査

```
YouTubeで「マッチングアプリの写真」に関するコンテンツを調査して。

【調査してほしいこと】
1. 人気動画の分析
   - 「マッチングアプリ 写真」で検索
   - 再生回数TOP10の動画内容
   - どんなテーマが伸びているか

2. 視聴者のニーズ（コメント欄）
   - どんな悩みが書かれているか
   - 何を知りたがっているか

3. 撮影サービス紹介動画
   - プロ撮影サービスを紹介している動画
   - どんな訴求をしているか
   - Before/After系の反応

4. 自撮りテクニック動画（競合）
   - 「自分で撮る方法」動画の再生数
   - それでも難しいポイントは何か

5. コンテンツトレンド
   - 最近の動画の傾向
   - エンゲージメントの高いテーマ

【検索キーワード】
- マッチングアプリ 写真 撮り方
- マッチングアプリ モテる写真
- プロフィール写真 コツ

【出力】
c:\re\カメラサービスの比較\04_youtube.md に保存して。
最後に「ビジネスへの示唆」を3-5点まとめて。
```

## 🔴 ウィンドウ5: 統合分析（全調査完了後）

```
c:\re\カメラサービスの比較\ にある以下4つの調査結果を読んで、統合分析とビジネス戦略を立案して。

- 01_coconala.md
- 02_timeticket.md
- 03_twitter.md
- 04_youtube.md

【分析してほしいこと】
1. ニーズ・ウォンツの整理
   - 顕在ニーズ（言語化されている）
   - 潜在ニーズ（まだ気づいていない）
   - ペインポイント（最大の障壁）

2. ターゲットセグメント
   - 最も狙うべきターゲット像
   - 年齢、性別、状況、心理

3. 競合との差別化ポイント
   - 価格で勝負するか
   - サービス内容で勝負するか
   - 体験・付加価値で勝負するか

4. 価格戦略
   - 推奨価格帯
   - プラン構成（松竹梅など）

5. マーケティング戦略
   - どのチャネルで集客するか
   - 訴求メッセージ
   - 口コミ獲得の仕組み

6. サービス設計
   - 必須要素
   - 差別化要素
   - あったら嬉しい要素

【出力】
c:\re\カメラサービスの比較\summary.md に保存して。
戦略提案書の形式で、実行可能なアクションプランを含めて。
```

---

# 実行手順

1. Claude Codeを4つ（または5つ）のウィンドウで開く
2. 各ウィンドウで作業ディレクトリを `c:\re\カメラサービスの比較` に設定
3. 上記プロンプトをそれぞれコピペして実行
4. 4つ完了したら、統合分析（ウィンドウ5）を実行
