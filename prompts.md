# マッチングアプリ専門カメラサービス - 市場リサーチプロンプト集

## 使い方
各プロンプトを別々のClaudeウィンドウ（またはSDK）で実行してください。

## 🟡 Agent 1: ココナラ調査

```
あなたはマーケットリサーチャーです。

【タスク】
ココナラで「マッチングアプリ用プロフィール写真撮影サービス」の市場調査を行ってください。

【調査項目】
1. 競合サービス一覧（上位15-20件）
   - サービス名、価格、販売実績数

2. 価格帯分析
   - 最安値〜最高値
   - ボリュームゾーン（最も売れている価格帯）

3. サービス内容の比較
   - 撮影枚数
   - レタッチ・加工の有無
   - ロケーション（スタジオ/出張/データ納品のみ）
   - 追加オプション

4. レビュー分析（★4以上と★3以下を比較）
   - 高評価の共通点
   - 低評価・クレームの傾向
   - ユーザーが本当に求めているもの

5. 売れてる出品者の特徴
   - プロフィールの書き方
   - サムネイル画像の傾向
   - 訴求ポイント

【検索キーワード例】
- マッチングアプリ 写真
- プロフィール写真 撮影
- マッチングアプリ 写真 男性
- Pairs 写真

【出力形式】
マークダウン形式で、具体的な数字・事例を含めて記載。
最後に「ビジネスへの示唆」を3-5点まとめる。
```

## 🟢 Agent 2: タイムチケット調査

```

5. レビューから見るユーザーニーズ
   - 満足ポイント

   - 「こうしてほしかった」の声

- プロフィール撮影
- 恋活 写真

## 🔵 Agent 3: Twitter/X調査

```
あなたはマーケットリサーチャーです。

【タスク】
Twitter/Xで「マッチングアプリの写真」に関するユーザーの生の声を調査してください。

【調査項目】
1. ユーザーの悩み・不満（検索して収集）
   - 「マッチングアプリ 写真 撮れない」
   - 「マッチングアプリ 写真 難しい」
   - 「マッチングアプリ いいね こない 写真」
   - 「自撮り 苦手」

2. 撮影サービス利用者の声
   - 「プロフィール写真 撮ってもらった」
   - 「マッチングアプリ 写真 プロ 頼んだ」
   - 満足/不満の声

3. 潜在ニーズの発掘
   - 「〇〇してくれるサービスあったら」
   - 「写真さえ良ければ」
   - 「誰か撮ってほしい」

4. ターゲット像の把握
   - どんな人が困っているか
   - 男性/女性の悩みの違い
   - 年齢層の傾向

5. 心理的障壁
   - なぜサービスを使わないのか
   - 恥ずかしさ、価格、その他

【出力形式】
マークダウン形式で、ツイートの傾向・要約を含めて記載。
（個人情報は含めない）
最後に「ビジネスへの示唆」を3-5点まとめる。
```

## 🟣 Agent 4: YouTube調査

```

【タスク】
YouTubeで「マッチングアプリの写真」に関するコンテンツを調査してください。

【調査項目】
1. 人気動画の分析
   - 「マッチングアプリ 写真」で検索
   - 再生回数TOP10の動画内容
   - どんなテーマが伸びているか

2. 視聴者のニーズ（コメント欄分析）
   - どんな悩みが書かれているか
   - 何を知りたがっているか
   - Before/After動画への反応

3. 撮影サービス紹介系コンテンツ
   - プロの撮影サービスを紹介している動画
   - どんな訴求をしているか
   - 視聴者の反応

4. 自撮りテクニック系（競合コンテンツ）
   - 「自分で撮る方法」動画の再生数
   - それでも難しいポイントは何か
   - サービス利用への導線になりうるか

5. インフルエンサー/発信者
   - このジャンルで影響力のある人
   - コラボ可能性

【検索キーワード例】
- マッチングアプリ 写真 撮り方
- マッチングアプリ モテる写真
- プロフィール写真 コツ
- Pairs 写真

【出力形式】
マークダウン形式で、具体的な動画例・数字を含めて記載。

## 🔴 統合分析（全調査完了後）

```
あなたはビジネス戦略コンサルタントです。

以下の4つの市場調査結果を統合し、マッチングアプリ専門カメラサービスのビジネス戦略を立案してください。

【調査結果】
（ここに4つの調査結果を貼り付け）

【分析項目】
1. ニーズ・ウォンツの整理
   - 顕在ニーズ（すでに言語化されている）
   - 潜在ニーズ（まだ気づいていない）
   - ペインポイント（最大の障壁）

2. ターゲットセグメント
   - 最も狙うべきターゲット像
   - 年齢、性別、状況、心理

3. 競合との差別化ポイント
   - 価格で勝負するか
   - サービス内容で勝負するか
   - 体験・付加価値で勝負するか

4. 価格戦略
   - 推奨価格帯
   - プラン構成（松竹梅など）

5. マーケティング戦略
   - どのチャネルで集客するか
   - 訴求メッセージ
   - 口コミ獲得の仕組み

6. サービス設計
   - 必須要素
   - 差別化要素
   - あったら嬉しい要素

【出力形式】
戦略提案書の形式で、実行可能なアクションプランを含めて記載。
```
